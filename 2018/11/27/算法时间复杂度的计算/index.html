算法时间复杂度的计算

Nov 27, 2018 | 数据结构与算法

一般用大写O()来表示算法的时间复杂度写法，通常叫做大O记法。
一般情况下，随着n的增大，T(n)增长最慢的算法为最优算法。

大O推导法：
用常数1取代运行时间中的所有加法常数
在修改后的运行函数中，只保留最高阶项
如果最高阶项存在且不是1，则去除与这个项相乘的常数

示例：

它的执行次数f(n) = 1 + n + n^2 + n(n+1)/2,根据推导大O阶的方法，最终它的时间复杂度为：O(n^2)



时间复杂度所耗费的时间是：
O(1) < O(logn) < O(n) < O(nlogn) < O(n2) < O(n3) <O(2n) < O(n!) <O(nn)
